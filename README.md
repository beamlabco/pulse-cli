# pulse-cli

## Build

- `python3 setup.py sdist bdist_wheel`
- `pip3 install dist/pulse-0.2-py3-none-any.whl`

Note: change the filepath to latest build.

## Local Installation

`pip3 install -e .`

## Installation

Install using `pip install git+https://github.com/yourusername/repository.git@v0.3`.
